---
description: 웹툰 추천 - 입력한 웹툰과 유사한 작품을 키워드/톤 매칭으로 추천합니다
---

사용자가 "$ARGUMENTS" 웹툰과 유사한 웹툰을 찾고 싶어합니다.

다음 단계를 수행하세요:

1. `python3 scripts/recommend.py "$ARGUMENTS" --top 10 --verbose` 명령을 실행하세요.

2. 결과를 보기 좋게 정리해서 사용자에게 보여주세요:
   - 기준 작품의 장르, 태그, 분위기 정보
   - 추천 결과를 테이블 형태로 (순위, 제목, 점수, 분위기, 핵심 유사 포인트)
   - 상위 3개에 대해 간단한 추천 이유 설명
   - 각 작품의 URL 링크 포함

3. 약칭으로 검색해도 됩니다 (예: "나혼렙" → "나 혼자만 레벨업")

4. 사용자가 특정 작품에 대해 더 알고 싶어하면 해당 webtoon 폴더의 md 파일을 읽어서 상세 정보를 제공하세요.
